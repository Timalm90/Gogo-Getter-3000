The Fantastic Gogo-Getter 3000
Overview

The Fantastic Gogo-Getter 3000 is a console-based C# application developed as an object-oriented programming exercise. The program simulates a simple vending machine system where the user can purchase collectible items called “Gogo’s,” manage their in-game currency, and interact with a character named Niklas.

The application includes branching logic, conditional storytelling, and multiple outcomes based on user decisions and collected items.
