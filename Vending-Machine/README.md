# Gogo-Getter 3000

A fun C# console application - a vending machine game where you collect Gogos!

## Requirements

- .NET 10.0 SDK or later

## Installation

```bash
dotnet restore
```

## Running the Application

```bash
dotnet run --project "Gogo-getter 3000.csproj"
```

Or if you have the alias set up:

```bash
dotnet run
```

## Project Structure

- `Program.cs` - Main entry point
- `VendingMachine.cs` - Vending machine logic
- `Inventory.cs` - Product inventory management
- `Product.cs` - Product definition
- `User.cs` - User/player management
- `Gogo-getter 3000.csproj` - Project file

## Build

```bash
dotnet build
```

## Clean

```bash
dotnet clean
```
